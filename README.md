# Female Autism Research Daily Report

女性自閉症研究文獻日報 — 每日自動搜尋 PubMed 最新女性自閉症相關文獻，由 AI 彙整分析，產生繁體中文摘要日報。

## 主題涵蓋

- 偽裝與補償（Camouflaging & Compensation）
- 診斷差異與篩檢工具
- 心理健康（憂鬱、焦慮、自傷、自殺）
- 飲食障礙與營養（ARFID、飲食障礙、腸腦軸）
- 生殖健康（月經、懷孕、更年期）
- 神經生物學與性別差異
- 生活經驗、歧視與社會模式
- 教育與就業

## 技術架構

- **資料來源**: PubMed E-utilities API
- **AI 模型**: GLM-5-Turbo（fallback: GLM-4.7 → GLM-4.7-Flash）
- **部署**: GitHub Pages
- **排程**: GitHub Actions（每日台北時間 21:05）
- **運行環境**: Node.js 24
